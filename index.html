The first step into Web development was to learn Html . Used to arrange webpages and house content of them, it was important for me to understand it and know how to use it correctly.

Over the last few months, I have worked to understand various elements in Html whether that be the “div”, “script” or "tag" elements, learning how to use these elements to convey information as well as learning the BEM model to structure the elements of the webpage.

CSS I have to say is my favourite language to use as a web developer the multitude of options allows for almost unlimited ways to style Html documents. After initially learning CSS I have also learnt SASS which just provides such an efficient way to code especially used with the BEM model.

I learnt many ways to style elements, but my favourite aspect is the way CSS can be used in conjunction to create sleek and professional-looking menus such as the one you are using now.

JavaScript is the language I have the least experience with when it comes to Front-End Development however it is the one, I am focusing the most on currently to improve my current capabilities. This site uses jQuery and over small uses of JavaScript to imporve overall user experience.

It is great for adding interactivity to a website, dynamic styling and can greatly improve the overall appearance.
